Case Study: MyFlix App

Overview

The MyFlix app is a movie discovery and management platform for movie lovers. It allows users to register, log in, browse movies, and manage their favorite movie lists. The app provides a seamless user experience by offering features like creating user profiles, adding movies to a favorites list, and storing personal preferences like date of birth and email.

This case study explores the key problems the app solves, the design approach I took to build it, and how I used modern technologies to solve those problems in an intuitive, user-friendly way.

The Issue I will address with this app

Before MyFlix, many movie enthusiasts faced the following challenges:

  • Disorganized Movie Recommendations: Users needed a better way to find and store their favorite movies in a list they can refer back to.
  • Lack of User-Centric Features: Many platforms didn’t provide a personalized experience to save and manage favorite movies.
  • Complex User Authentication: Logging into platforms and managing user data was often confusing and time-consuming.

The Solution: MyFlix App

To address these issues, I designed MyFlix as a simple but effective platform that offers:

  • User Profiles: Users can create personal profiles with their username, email, and date of birth.
  • Movie Management: A personalized favorite movie list, where users can add and remove movies based on their interests.
  • Seamless Authentication: Secure login process that uses JWT (JSON Web Token) authentication for user sessions.

Key Features & How They Solve the Problems

User Authentication

Problem: Users had to manually input their credentials every time they logged in.

Solution: The app uses JWT authentication, allowing users to securely log in once and maintain a session, reducing the friction of logging in repeatedly.

User Profile Management

Problem: Lack of an easy way to manage user profiles and access personalized data.

Solution: MyFlix allows users to store and update their personal information (username, email, birth date) in one place.

Favorite Movies List

Problem: Movie enthusiasts faced difficulties organizing and keeping track of their favorite movies.

Solution: MyFlix allows users to add and remove movies from their favorite list, ensuring they can always keep track of the movies they love.

Search & Browse Movies

Problem: Finding relevant movie recommendations could be overwhelming.

Solution: MyFlix provides an easy-to-use search functionality where users can explore movies by title, genre, or director.

Backend

The backend uses the following technologies:

  • Node.js & Express: For handling user requests, authentication, and API communication.
  • MongoDB: Used for storing user and movie data. Its NoSQL nature makes the app scalable and flexible.
  • JWT Authentication: For secure user authentication and maintaining sessions.

Challenges Faced and Solutions

  • Handling Asynchronous Data Requests: I used React’s state management to handle asynchronous requests gracefully.
  • Ensuring Secure Authentication: Implementing JWT authentication ensured secure user login and session management.
  • Managing User-Favorite Lists: Real-time synchronization of users' favorite movie lists was achieved through backend communication.
  • Scalability & Performance: MongoDB was used to ensure the app can scale efficiently as more users and movies are added.

Technology Stack

Frontend: React.js, React Router, React Bootstrap.

Backend: Node.js, Express, MongoDB, JWT.

Impact and Results

  • Increased User Engagement: Personalized movie management features led to longer user interactions.
  • Improved User Retention: The app’s personalized experience increased user retention.
  • Seamless User Experience: Intuitive design and fast authentication made it easy for new users to get started.

Conclusion

MyFlix solves key problems in movie management by providing a simple but powerful platform for managing favorite movies. By using modern technologies like React, Node.js, JWT, and MongoDB, I was able to create a scalable, secure, and user-friendly application that delivers real value to movie lovers everywhere.

Future improvements include adding administrative users to clean up data and fix security flaws to ensure secure access to user data.
